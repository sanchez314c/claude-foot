# Code of Conduct

## Our Standards

ClawdFoot is a small open source project. We keep things simple: be respectful, be constructive, and focus on making the tool better.

**Good behavior:**
- Giving and receiving feedback gracefully
- Helping newcomers with setup or theme creation
- Filing clear bug reports with reproduction steps
- Keeping discussions focused on the project

**Not acceptable:**
- Personal attacks or harassment of any kind
- Trolling, insults, or derogatory comments
- Publishing others' private information
- Any conduct that would be considered inappropriate in a professional setting

## Scope

This applies to all project spaces: GitHub issues, pull requests, discussions, and any public channels where you represent the project.

## Enforcement

Project maintainers will remove, edit, or reject contributions that violate this code. Repeated or serious violations will result in a ban from the project.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainer at GitHub: [@sanchez314c](https://github.com/sanchez314c).
